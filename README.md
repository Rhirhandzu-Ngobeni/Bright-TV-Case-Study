Overview

This project analyzes Bright TV’s user behaviour and viewership patterns to support the CEO’s objective of *growing the subscription base*. The analysis focuses on understanding user engagement, content preferences, and consumption behaviour across demographics, time, and channels.

The dataset includes:
- User profile information (demographics)
- Viewership transaction data (session-based consumption)# Bright-TV-Case-Study

- Business Objective

The CEO aims to increase Bright TV’s *subscription base* by:
- Understanding user engagement patterns
- Identifying drivers of content consumption
- Detecting low-engagement periods
- Providing data-driven recommendations for growth

 Key Analyses Performed

1. User Engagement Analysis
- Active vs inactive subscribers
- Engagement distribution across gender, race, and province

2. Channel Performance
- Most viewed channels by share of consumption
- Top-performing content categories (sports, entertainment, etc.)

3. Temporal Consumption Patterns
- Viewership by day of week
- Viewership by time of day (morning, afternoon, evening, late night)

4. Content Duration Analysis
- Average viewing duration per channel
- Identification of high-retention content

Key Insights

- *Sports content dominates engagement*, led by SuperSport Live Events and ICC Cricket World Cup.
- *Evening hours and mid-to-late week periods* represent peak consumption times.
- *Young adults form the core user base*, with strong regional engagement in Gauteng and Western Cape.
- Significant *data quality gaps exist in demographic fields*, especially “unknown” categories.

- Recommendations

- Invest in *sports rights and live event content* to drive acquisition and retention.
- Optimize content scheduling around *evening peak hours and weekend engagement periods*.
- Improve *user data collection and profiling* to reduce unknown demographic entries and improve targeting.
- Develop strategies to boost engagement during *low-activity periods (Mondays and late nights)*.

  Tools Used

- SQL (Databricks)
- Excel (Pivot Tables & Visualization)
- Data Cleaning & Transformation
- Descriptive Analytics

  Conclusion

This analysis shows that Bright TV’s growth opportunity lies in leveraging *sports-driven engagement, optimizing **prime-time scheduling, and improving **data-driven personalization strategies* to convert viewers into long-term subscribers.
